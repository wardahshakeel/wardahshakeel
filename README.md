## 📁 Featured Projects

🔹 **[Telecom Customer Churn Prediction](https://github.com/wardahshakeel/customer-retention-prediction)**
End-to-end ML pipeline predicting churn on the IBM Telco dataset (7,043 records). Covers EDA, feature engineering, model comparison (Logistic Regression vs. Random Forest), class imbalance handling, and threshold tuning — final model reaches 0.84 ROC-AUC with 79% recall on churned customers, plus concrete retention recommendations.

🔹 **[GA4 E-Commerce Funnel Analysis](#)**
SQL/BigQuery analysis of the public GA4 e-commerce dataset, using CTEs and conditional aggregation to reconstruct the user funnel (page view → purchase) and quantify conversion and drop-off at each stage.
